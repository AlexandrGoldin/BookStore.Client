# BookStore.Client
Frontend of the Client-Server my pet project of the online sopping "BookStore" for PC. Developed in Visual Studio Code on the React.js/Redux.
_________
## Stack:
* React.js
* Redux
* Redux-Thunk
_____________________

Главная страница _BookStore_ со статистикой, фильтрами, корзиной покупок, кнопкой продолжить, формой заказа, и кнопкой _Проверить_
